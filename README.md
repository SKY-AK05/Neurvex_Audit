# NeuroMark Audit Tool

Web-based neurodiversity inclusion self-assessment tool built for Orchvate.

## What it does
- 40-question audit across 8 sections
- Automatic scoring and maturity level assignment
- Admin dashboard to review, edit, and send personalised email reports
- Backed by Azure Functions + PostgreSQL + Azure Communication Services

## Quick Start
See [docs/SETUP.md](docs/SETUP.md) for full setup instructions.

## Stack
- Frontend: HTML + CSS + Vanilla JS (static files)
- Backend: Azure Functions (Python)
- Database: Azure PostgreSQL Flexible Server
- Email: Azure Communication Services
